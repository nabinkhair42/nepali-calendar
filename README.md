# Nepali Calendar

A small macOS menu bar app for the Bikram Sambat calendar.

Nepali Calendar keeps today’s BS date one click away, with Nepali/English labels,
festival details, public holidays, and light/dark mode support. It is built for
people who use the Nepali calendar every day but do not want a full calendar app
open on their desktop.

## Install

Download the latest app from:

https://calendar.nabinkhair.com.np

Or install from Terminal:

```bash
curl -fsSL https://calendar.nabinkhair.com.np/install.sh | bash
```

The installer downloads the current disk image, verifies it, copies the app into
`/Applications`, and launches it.

## Privacy

The app has no accounts, no analytics SDKs, and no tracking scripts. It stores
only a local festival cache and your language preference.

## Requirements

- macOS 14 or later
- Menu bar access
